单选与多选

# 使用自定义标记,多选 <m-check> , 单选 <m-radio>


# 属性 checked 尝试

选中返回true,否则返回false

设置为true时,选中,否则取消选中.

# 属性 onClicked(fn)

点击切换后执行方法.(仅单选按钮)

# 标签属性 tag

设置一个标题

# 单选

同一个父级下的多个单选框,设置相同的name属性,实现单选.已选中的可以取消.

# 选中状态,大小,颜色

添加 .checked样式,会展现选中状态,使用工具样式调整颜色和大小

// m-check多选,m-radio单选
<m-radio tag="单选框2"></m-radio>
<m-check class="checked" tag="多选框1"></m-check>

// tag:设置标题
<m-check class="checked" tag="多选框1"></m-check>

// 获取选中状态
let check1 = document.getElementById('checkbox1');
check1.checked // true / false
check1.checked = true // true选中,false取消选中

// 执行方法
check1.onClicked = (radio) => {
  // radio参数就是this对象
  radio.getAttribute('tag');
}

// 单选
<m-radio tag="单选框1" name="radiogroup1" ></m-radio>
<m-radio tag="单选框2" name="radiogroup1" ></m-radio>

// 颜色和大小,选中状态
<m-check class="red checked " tag="多选框1"></m-check>
<m-radio class="text-lg" tag="单选框2"></m-radio>

// 禁用
<m-radio disabled tag="单选框2"></m-radio>
