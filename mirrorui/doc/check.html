单选与多选

使用自定义标记,多选 <m-check> , 单选 <m-radio>

// m-check多选
<m-check class="checked" tag="多选框1"></m-check>
// m-radio单选
<m-radio tag="单选框2"></m-radio>

单选实现

同一个父级下的多个单选框,设置相同的name属性,实现单选.已选中的可以取消.

属性

checked 选中返回true,否则返回false.设置为true时,选中,否则取消选中.

// 获取/设置checked
let check1 = document.getElementById('checkbox1');
let isChecked = check1.checked; // true / false
check1.checked = true; // true选中,false取消选中

如果需要设置默认选中,在标记上添加checked样式

<m-check class="checked" tag="默认选中"></m-check>

tag 设定标题

// tag,设置标题,在标记上使用
<m-check tag="多选标题"></m-check>
<m-radio tag="单选标题"></m-check>
// 
check1.tag = '选项框'; // get set

disabled 禁用控件,加disabled属性

<m-radio disabled tag="单选框2"></m-radio>

事件

onClicked(fn) 点击切换后执行方法.

radio1.onClicked = (radio) => {
  // radio或者check参数就是this对象
  let getTag = radio.getAttribute('tag');
}
