标签页

容器组件,选择标签卡,显示对应页面.标签卡和页面由索引位置关联.

自定义组件,删减了添加选项卡相关功能,简洁了样式,主要用于展示内容.

<m-tabs></m-tabs>

// 内部分为标签块和内容块2部分
// 标签框
<div class="tabs-header">
  <span class="tabs-label active">苏轼</span>
  ...
</div>

// 内容框
<div class="tabs-content">
  <div class="tabs-panel active">
    <h5>饮湖上初晴后雨二首·其二</h5>
    <p>水光潋滟晴方好</p>
  </div>
  ...
</div>

// 可以在第一个标签和对应的内容加上active样式,默认显示

属性

height css合法的height值.在m-tabs上设置容器高度,默认320px.内容超过高度时,出现竖直滚动条.

// height: 高度设置
<m-tabs height="200px" ></m-tabs>

方法

tabs.activeTab(index)激活标签.index:标签索引.

// 激活第一个标签
tabs.activeTab(0)
