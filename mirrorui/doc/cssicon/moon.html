实现过程

月牙可以看成是一个圆的一部分,所以先做一个圆.这个用border-radius:50%实现.就是圆角边框.


  display:inline-block;
  border-radius :50%;
  width:1em;
  height:1em;

边框阴影也会跟随圆角是一个圆形.调整阴影的x,y值改变月牙的大小和位子.

x值增大时,阴影再右侧增大,月牙变大.y值增大时,阴影向下倾斜,月牙呈现顺时针旋转


  box-shadow: 0 0 0 0;
  box-shadow: 10px 0 0 0;
  box-shadow: 15px 0 0 0;
  box-shadow: 10px 5px 0 0;
  box-shadow: 10px 10px 0 0

由于阴影不占用元素面积,月牙部分可能会盖到其它元素,可以设置margin-right抵消.

阴影的颜色跟随color属性,大小可以用em单位.这样便于支持大小和颜色,完整css规则.


   <i class="ico ico-moon"></i>

  .ico-moon
      display: inline-block
      width: 1em
      height: 1em
      margin: 0 .9em .1em 0
      border-radius: 50%
      background-color: transparent
      box-shadow: .4em .1em 0 0
