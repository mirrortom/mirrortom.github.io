列表

使用ul,ol标记,加list类.

根据原有的ul,ol的样式,稍微调整了li的上下padding.在嵌套时的padding-left.另外,加了数值和字母序列

无序列表

包含5种图案风格,使用了emoji图案,跟随字体颜色.

<ul class="list">

// 形状 圆点(默认),方块,圆心,三角形,五角星
<ul class="list disc">
<ul class="list square">
<ul class="list cir">
<ul class="list triangle">
<ul class="list star">
圆点方块 ■圆心 ○三角形 ▲五角星 ★
  • 图案标记列表支持5种图
  • 圆点,默认
  • 空心圆
  • 正方形
  • 三角形
  • 五角星
  • 支持主题颜色

有序列表

4种风格,可选是否计算嵌套.不支持颜色主题.

 <ol class="list">

// 数值,大写字母,小写字母,中文
<ol class="list num">
<ol class="list abc">
<ol class="list abc-l">
<ol class="list zh">
// 要计算嵌套,在风格样式后加上s
<ol class="list nums">

不计算嵌套,每个list都从新开始计数,可以多种风格一起使用

  1. 支持4种风格
  2. 字母
    1. AAAAAAAAAAAA
    2. BBBBBBBBBBBB
    3. CCC
  3. 中文
    1. 一二三
    2. 四五六
  4. 小写
    1. cccccccccccc
    2. dddddddddddd

计算嵌套(样式名字后面带上s),从最外层的list开始计数,只能用同一风格,嵌套的list不必再加风格类

  1. VIP会员全部消费7折
  2. 普通会员9.5折
    1. 资深,8折
    2. 贵宾7.5折
      1. 随机优惠券
        1. 5圆
        2. 20圆
        3. 50圆

# 主题颜色适用在形状上.编号无效.
