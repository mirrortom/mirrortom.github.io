# 数值编号,字母编号样式

嵌套的序列使用了css的 "counter-reset" "counter-increment" "counters()" 这三个属性和方法

  // 方块 , 圆
  <ul class="list square">
  <ul class="list cir">

  // 主题色
  <ul class="list primary">

  // 数值 , 字母
  <ul class="list num">
  <ul class="list abc">
  
  // 嵌套
  <li>
    优惠券
    <ol class="list abc">
      <li>酒10元</li>
      <li>肉10元</li>
      <li>免5元</li>
    </ol>
  </li>

  注意:嵌套的ul,ol,也要加上 class="list num|abc" 样式
