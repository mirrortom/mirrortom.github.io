列表

使用ul,ol标记,加list类.

根据原有的ul,ol的样式,稍微调整了li的上下padding.在嵌套时的padding-left.另外,加了数值和字母序列

# 无序列表,包含4种风格,支持颜色主题

<ul class="list">

// 形状 方块,圆,三角形
<ul class="list disc">
<ul class="list square">
<ul class="list cir">
<ul class="list triangle">
  • 面包
    • 白面包
    • 黑面包
  • 米饭
    • 大米
    • 香米
  • 馒头
    • 白面馒头
    • 黄米窝头

# 有序列表,3种风格,可选是否计算嵌套.不支持颜色主题.

 <ol class="list">

// 数值,大写字母,中文
<ol class="list num">
<ol class="list abc">
<ol class="list zh">
// 要计算嵌套,在风格样式后加上s
<ol class="list nums">

不计算嵌套,每个list都从新开始计数,可以多种风格一起使用

    2. 猪脚
  2. 牛肉
    1. 牛排
    2. 牛肚
  3. 羊肉

    2. 羊腿
