菜单条

用于手机,模仿app菜单页面,右侧有个箭头的那种菜单

容器使用flex布局,两端对齐.直接子项目为2个,分别在左右两侧.

标题和箭头
🍄 带图标标题
标题和副标题
关闭数据
🚯 带图标标题和副标题
禁止丢垃圾 

    // 标题和箭头,直接作为子项目
    <div class="menubar">
      <span>标题和箭头</span>
      <span class="arr"></span>
    </div>

    // 带图标的,需要将标题和图标再套一层DIV
    <div class="menubar">
      <div>
        <span class="iconface">🍄</span>
        <span>带图标标题</span>
      </div>
      <span class="arr"></span>
    </div>

    // 箭头和副标题也是套一层DIV
    <div class="menubar">
      <span>标题和副标题</span>
      <div>
        <span class="">关闭数据</span>
        <span class="arr"></span>
      </div>
    </div>
