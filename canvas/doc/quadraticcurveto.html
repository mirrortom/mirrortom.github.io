quadraticCurveTo(cpx, cpy, x, y)

新增二次贝塞尔曲线路径的方法.它需要2个点. 第一个点是控制点,第二个点是终点. 起始点是当前路径最新的点,当创建二次贝赛尔曲线之前,可以使用 moveTo() 方法进行改变.

  ctx.quadraticCurveTo(cpx, cpy, x, y);

参数

Number 控制点的 x 轴坐标.

Number 控制点的 y 轴坐标.

Number 终点的 x 轴坐标.

Number 终点的 y 轴坐标.


测验

二次贝塞尔曲线,起(50,20)控(230,30)终(50,100)
