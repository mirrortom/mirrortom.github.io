bezierCurveTo(cpx, cpy, cpx1, cpy1, x, y)

创建3次贝塞尔曲线路径,前两个是控制点坐标,最后是结束点坐标,起点是上次moveTo()或lineTo()的坐标

  ctx.bezierCurveTo(cpx, cpy, cpx1, cpy1, x, y);

参数

cpx Number 控制点1的 x 轴坐标.

cpy Number 控制点1的 y 轴坐标.

cpx Number 控制点2的 x 轴坐标.

cpy Number 控制点2的 y 轴坐标.

x Number 终点的 x 轴坐标.

y Number 终点的 y 轴坐标.


测验

三次贝塞尔曲线,起(20,20)控1(60,140)控2(280,40)终(260,140)
