strokeText(text, x, y [,maxWidth])

绘制描边文本内容

  ctx.strokeText(text, x, y [,maxWidth])

参数

string 要绘画的文本内容

Number 文本左上顶点x坐标

Number 文本左上顶点y坐标

Number 文本最大宽度,如果设定了,当文本超过设定宽度时,文本将会缩放适应宽度


测验

写入一段文本hello world (50,75),可见方法只对文字描边,与fillText()不同

写入一段文本hello world (50,75),宽度为75px.可以看见,对比上图,文字被压缩了,因为超过了75px宽度
