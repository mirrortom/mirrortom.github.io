依据亚伯拉罕·马斯洛的需求理论,人通过化妆得到一个光鲜的外表,这属于哪一种需求呢.属于生理需求,因为好看的外表,容易吸引异性.

但也不是全部这样,也可以是尊重的需求,因为"人靠衣妆",通过化妆得到被尊重的需求.大多数情况下,一个衣妆精美的人,会被另眼相看.

还可以是自我实现,通过化妆表现自己所期待的自我.

化妆是一个手段,为何而化妆,不同的人出发点不一样.有点像吃饭,记得有句话说,有的人活着是为了吃饭,有的人吃饭是为了活着.这两种情况表现的意义完全不同.

不管处于那个需求阶段的人,都有化妆的.就像吃饭,是一个刚需要求.吃的好坏,妆的好坏不同.

前两天向友人请教了一下化妆品的一些认知,今天做一下记录.以下是请教的大致内容.

上海人,使用国产品牌化妆品用的多,还是国外品牌?

// 上海人用国产品牌的主要是一些老年人,其它都是国外品牌的.

// 主要是市面上的国货比较少,国内化妆品公司竞争不过外资,有的关门不做了,有的被收购.导致国货太少了,商场里大部分都是国外品牌了.

// 化妆品作为快消品,营销是关键因素.国内公司这方面比较差,再加上财力也不够,所以难以建立好的品牌形象.

那么,不是质量或者技术方面问题,是品牌认可不够高的问题?

// 可以看看化妆品公司的财报,毛利非常高,营销费用占了一大部分,研发费用基本上可以忽略了.

我找了上海家化的近期财报看了这几个数据,确实如此.大概研发3千万,销售费用12亿,毛利10亿.从毛利看,这有点像IT业,研究出一个产品,就可以复制了.

从原材料组成看,化妆品起作用的成分占5%不到,其它是水.研发费用不高,可以认为技术仿制的门槛不高.那么人都可以做的话,那么品牌成了关键.

大多数情况下,谁的品牌知名度高,就买谁的,商场货位也愿意摆放这个品牌.一个好的品牌,可能潜在的被贴上几个标签.技术高,质量好,服务好,大品牌有面子.

一个不太恰当的比喻,一个人有钱了,不管怎么来的.在他说同样励志的话时,要比那个没钱的人讲出来,效果要好,听众更有兴趣,可能是因为能够"反证".

化妆品的利润比起同为快消品的食品如何?

// 食品比化妆品周转更快,毛利比化妆品低很多,不过由于周转快,在同样大小规模下比较,食品总利润可以比化妆品高,不过化妆品赚钱比食品要轻松多了.

// 收智商税(品牌价值)的模式当然更轻松.

外资化妆品关税?

// 关税好像是按奢侈品类目交的,30%左右.还有增值税和消费税.为了降低成本,外资采用大包装进来,再到国内分装的办法.一大袋化妆品,

// 十几公斤装的,分装成小瓶的零售,可能就是超市那种几十几百毫升的,但是价格就放大了几十倍到百倍.这种从大包装到零售成品的办法,可以少交税.

// 如果直接卖成品(价格高),那么交的税也多.所以先进大包(价格低),少交税,再分装零售,把价格调高.这个就是有巨大利润了.

我看,是比较在理了,为了避税增加利润,这是个不错的策略.

那么,化妆品的品牌是关键,其实也可以感受到.在大商场里,在地铁里,在城市各显眼的广告位,时常都是国外化妆品品牌.花巨资投入广告费,

才打造出品牌形象.试想,如果一段时间某个知名化妆品品牌不打广告了,那么有可能在人心目中的地位就下降了.因为一段时间没有听说,就可能会想:

好久没有见这个品牌了,是不是不行了?所以,需要不停的花钱做广告,越是高级品牌,花的广告钱就越多,它不能在低价的广告平台做,

会应影响它的品牌.(参考.吴军.浪潮之巅)

显然,老外比国人要明白这个道理.所以铺天盖地的广告,甚至货都没有到,国人就知道有这样一个老外品牌,跃跃欲试.市场被外资抢了,为数不多的,

认真做产品的国货立即就困难了,不知道为什么几十年上百年的深入人心老牌子,怎么一下子就被外资品牌打败,利润挤压的所剩无几.等到国人明白过来,又为时晚矣,

市场已经被外资占领了,随着新一代年轻人长大,他们脑海里都是外资品牌了.因为在他们年少时,看到的都是这些外资品牌广告,现在有了钱,首选外资.

如果说仓禀实而知礼节,在更多的人吃饱饭以后,化妆的需求也更多了.国内市场巨大,对化妆品的需求也巨大,化妆品类似于IT行业,存在垄断,赢者通吃的局面,虽然不像IT也那么极端.

人们总是在期待有新秀出来,不过在品牌认知度已经如此强大的外资面前,杀出一匹国货黑马,不是容易的事.国货化妆品在国人心中依然是第二梯队,第二选择,这个观点还要持续...
