看星星的夜晚,回忆总是远去,笑声仿佛昨日.倾尽关怀,让心灵归宿.穿越时空,在月夜下沉思....

学习,主动了.为了技能生存的学习.从年前开始,已经将近四个月.既定目标3个月完成学习并做好项目,业已无法实现,有很多时间安排在了错误的地方.没有抓住重点,虽然没有浪费,但却没有效率.本以为自己做了充足的准备,但还是发现自己在新理论和概念面前根本无法快速理解和接收.事实证明我在这方面并没有天赋,有的只是兴趣,或许还有长时间找不到好的学习方法的原因吧.

今天做这个总结,回顾一下历程.从我接触计算机开始,接触了方方面面,开始是在学校,说是个先进的东西,在黑屏上打字学五笔,和学校的其它课一样,开始有兴趣学习,不久便被弄得了无生趣,机器两人一台,一星期上两节课,我几乎摸不到机器.没有天赋的我,自然没有抓住这个微小的机会.如果那时我看到计算机今天的样子,会觉得非常可惜,我曾经离它这样的近.虽然学校的计算机课很快就消失了,如同政府的面子工程.

但在几个同学的影响下,我还是对它神奇不能忘怀.久久地想着.后来有机会得到一个学习机,我一有机会便会玩.主要吸引我的是游戏,虽然有打字和BASIC编程,但我只写过十几行程序,还是照着同学写在纸上的输入的,虽然是十几行的加法程序,但却成功了.我当时惊奇不已.不过仅此一次,最后吸引我的还是游戏.并且遭到反对.印象之中,我的兴趣方面很少有志同道合者,身边的人都是反对的,所以我保持沉默.学习机玩过了,计算机普及了,学电脑成了热潮.

我依然热衷于计算机的游戏,不过也不在行.唯一花功夫的是学五笔,想起在学校老师罚我N次都学不会,这次却学了几天就会了,只是看了理论做了些实践练习之后就会了.感觉很神奇.计算机真的好玩.此时我对它还是我一无所知,不知道计算机里有程序,有硬件,计算机可以联网,还能做网站等等.我依然只知道用它玩游戏,后来还学会了用它上网,聊天.有很长一段时间是计算机陪我渡过的.给了忧忧郁郁的我许多快乐.从DOS时代386到win32/95/98/me/2000那些界面我都有个眼识,是个见识很多的门外汉.又是几年,我依然在计算机上得到许多快乐.我的人生还是只有卑微的论调.人生如梦现实如洗,唯有内心深处留有一个希望.从不会提起.那是因为关怀..

计算机在快速的发展,而我仍然拿它当作快乐的伙伴,仍然对它不了解.现实终要面对,不管有没有做好准备,有没有醒悟.我开始学习计算机.有一个机会,我学习计算机了,自己学习,不喜欢老师,只有一些资源.凭着一点微弱的悟性,我学习得很吃力,但也很卖力.首次拆开机器,查看结构,再组装计算机,多次实践后学会了所谓的装机,就是将各组件拼接在一起.接着安装操作系统,此时还知道了操作系统不是微软一家,至少还有苹果和LINUX,我没装过苹果的,听说是和硬件一起卖的.

通过资料,我还接触了一些语言,VBVC,一些软件,ACCESS数据库,易表等.虽然都还没入门,按教程做了一些小程序,但却觉得这真是个好玩的东西,计算机真正吸引人的地方在编写代码让计算机按自己的想法工作.可惜,我却无法拿它谋生,我的学习程度太浅,不足以找到工作.此时,我对计算机的了解比以前多多了,我虽然还当它是快乐的伙伴,但兴趣已经不只是玩游戏了,偶尔也会写程序,做网页,测试性能,装多个系统.对它的兴趣随着了解它而增加了.后面的经历让我进一步了解计算机.计算机众多的应用背后有许许多多的技术支持,带给我快乐的游戏,上网,聊天都是应用程序工作的结果.

为此要了解要学习的东西太多了,操作系统,人机交互的接口,不可缺少的软件,它是怎么设计实现的.数据库,存放数据,海量的数据在这里,有名的数据库MYSQL SQLSERVER ORACLE DB2,使用SQL语言.程序语言,C C++ JAVE C# PERL等,WEB HTML CSS JAVASCRIPT XML 还有访问数据库的多种技术.网络服务器,IIS/APACHE,虚拟平台,JAVA VM ,.NET 等等.我这几个月学习的东西比前N年学的加起来都多,虽然有些是回顾,但也有创新,计算机发展得太快了,我多年了还在摆弄以前的机器,却发现现在机器硬盘接口早已是SATA的了.CPU也都是大风扇大散热器.笔记本计算机也普及了,平板电脑有了,触控操作非常便捷.有非常失落的感觉,先接触先过时,曾经和浪潮擦肩而过,曾经有机会踏上浪潮,而一切都没有发生,浪潮在前,我湿了衣服.原地不动.打在脸上的几滴,让我回忆感怀,重拾快乐.不见了98那简朴的色调,也没有了企鹅小巧的身影.增加了许多色彩,虽然我习惯简朴,但计算机仍是我快乐的伙伴.回忆到此,愿我终以计算机为伍.带给人给我的快乐.
