# 点击切换 list-style-type:"xxx"

    计数器

    有2个属性 "counter-reset" "counter-increment" 2个方法 "counters()" "counter()"

    ul{
// 定义一个名叫li-item的计数器
counter-reset:li-item; 
}
ul > li::before{
// 设置li-item计数器的步增为1,如果要为3,写成"li-item 3"
// 注意是写在ul的li的伪类上的,也就是说计数器对这个元素计算
counter-increment: li-item; 

// 这是个函数,参数1是计数器名字,参数2是加在数值后面的字符串,参数3是样式(默认是数值,还可以是字母"lower-latin")
content: counters(li-item,'.') '. ';

// 这个函数和上面的作用范围不同,它不会计数嵌套的层级
content: counter(li-item,'.') '. ';
}
