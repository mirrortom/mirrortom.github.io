线性渐变 | 径向渐变 | 圆锥渐变

简介

此文记录下css圆锥渐变的用法还有几个效果例子


    background-image:conic-gradient(red,yellow);

(来自MDN)颜色过渡围绕中心点旋转,而不是从中心辐射

圆锥渐变的路径是,围绕一个中心点,从一个起点角度开始,顺|逆时针旋转运动的走向发生渐变.

如果俯视这个图,像是一个圆锥体的平面投影,中心点就是圆锥的尖顶.不过,这更像是圆周运动啊,为什么不叫圆周渐变呢.

中心点

圆锥渐变中心点可设置: (at left top). 和径向渐变一样.为负值时,中心点不在盒模型上.默认center(盒模型中心).


    // at left top
    conic-gradient(at 60px -20px, red,yellow);
    conic-gradient(at 50% 80%, red,yellow);

旋转起点

旋转起点默认0deg,设置:(from 0deg). 中心点竖直向上为0度(钟表12点位置),或到盒模型上边的射线,为0度起点.同线性渐变一样,顺时针方向.


    // -90deg,从盒模型左边中线处开始
    conic-gradient(from -90deg, red,yellow,blue);

    // 从盒模型下边开始,中点与下边垂线处
    conic-gradient(from 180deg at 70% 70%, red,yellow,blue,purple);

位置

起止位置,可设置1个或者2个值,与线性渐变类似.用角度单位比较明了.


    // 4个颜色,3个渐变区间,均分360度.
    conic-gradient(red,yellow 120deg,blue 240deg,purple);

    // 2值法典型应用
    conic-gradient(from 90deg,red 0 90deg,yellow 90deg 180deg,blue 180deg 270deg,purple 270deg 360deg);

渐变(颜色)中点

在2个颜色间设置,类似线性渐变,该位置处,颜色处于过渡的中间值.


    // 默认值
    conic-gradient(red,orange,blue,green,purple);

    // 等效值 5个颜色,有4个渐变区间,均分360度,每个区间90度,第一个区间的中点是45度
    conic-gradient(red,45deg,orange,135deg,blue,225deg,green,315deg,purple);

示例

使用突变/叠加/重复做出图案.

微软图标

4色,每个占90度.


    conic-gradient(#f35325 90deg,#81bc06 90deg 180deg,#ffba08 180deg 270deg,#05a6f0 270deg)

统计饼图

很适合做饼图.


    border-radius: 50%;
    conic-gradient(red 0 30deg,orange 30deg 200deg,blue 200deg 220deg,green 220deg 260deg,purple 260deg 280deg,gray 280deg);
叠加

4个2色渐变叠加,每个显示一种颜色,另一个透明.旋转起点也错开.


    conic-gradient(from 30deg,red 0 30deg,transparent 30deg),
    conic-gradient(from 120deg,red 0 30deg,transparent 30deg),
    conic-gradient(from 210deg,red 0 30deg,transparent 30deg),
    conic-gradient(from 300deg,red 0 30deg,transparent 30deg);

圆锥重复渐变

使用函数repeating-conic-gradient()

纸风车


    repeating-conic-gradient(red 0 45deg,yellow 45deg 90deg);

3种渐变叠加


    repeating-conic-gradient(from -5deg,green 0 10deg,transparent 10deg 90deg),
    linear-gradient(-45deg,transparent 0 24%,red 24% 26%,transparent 26% 74%,red 74% 76%,transparent 76%),
    linear-gradient(45deg,transparent 0 24%,red 24% 26%,transparent 26% 74%,red 74% 76%,transparent 76%),
    repeating-radial-gradient(blue 0 2px,transparent 2px 12%);
